# LibraCore

LibraCore adalah aplikasi **Perpustakaan Digital berbasis Website** yang dikembangkan sebagai **Tugas Besar Praktikum Web 2025/2026** Fakultas Teknik Universitas Pasundan.  
Aplikasi ini menyediakan fitur pengelolaan buku digital, kategori, pengguna, serta laporan dalam format PDF dengan sistem autentikasi yang aman dan terstruktur.
